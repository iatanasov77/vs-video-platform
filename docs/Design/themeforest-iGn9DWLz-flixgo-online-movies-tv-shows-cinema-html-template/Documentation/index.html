Getting started

Unzip the archive and find out the folder www. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. /main — folder with main template.
  2. /admin — folder with admin template.

Main folder:

  1. /css — folder with css files.
  2. /icon — folder with favicons.
  3. /img — folder with image files.
  4. /js — folder with javascript files.

  5. index.html, index2.html — home page options.
  6. catalog1.html — catalog page style 1.
  7. catalog2.html — catalog page style 2.
  8. details1.html — details page style 1.
  9. details2.html — details page style 2.
  10. about.html — about page.
  11. actor.html — actor page.
  12. faq.html — FAQ page.
  13. pricing.html — pricing page.
  14. profile.html — profile page.
  15. contacts.html — contacts page.
  16. privacy.html — privacy page.
  17. signin.html — signin page.
  18. signup.html — signup page.
  19. forgot.html — forgot password page.
  20. 404.html — 404 page.

Admin folder:

  1. /css — folder with css files.
  2. /icon — folder with favicons.
  3. /img — folder with image files.
  4. /js — folder with javascript files.

  5. index.html — dashboard page.
  6. catalog.html — catalog page.
  7. reviews.html — reviews page.
  8. users.html — users page.
  9. comments.html — comments page.
  10. add-item.html — item page.
  11. edit-user.html — user page.
  12. signin.html — signin page.
  13. signup.html — signup page.
  14. forgot.html — forgot password page.
  15. 404.html — 404 page.

Upload the template files (inside the folder www) to server with the help of one of the FTP-clients.

HTML structure

In my template I use Bootstrap, if you don't know how to use it, here is a detailed information on its use.

The standard section has the following HTML structure:

<section class="section">
  <div class="container">
    <div class="row">
      <div class="col-12">
        <h2 class="section__title">Expected premiere</h2>
      </div>
      ...
    </div>
  </div>
</section>

Home

Section have a two different styles:


section with static background image

<section class="home home--bg">
  <div class="container">
    <div class="row">
      ...
    </div>
  </div>
</section>

section with carousel

<section class="home">
  <div class="container">
    <div class="row">
      <div class="col-12">
       <div class="hero splide splide--hero">
          <div class="splide__arrows">...</div>
          <div class="splide__track">...</div>
       </div>
      </div>
    </div>
  </div>
</section>

Sections

By default section have white background color, but you may change style:


section--bg – section with static background image

<section class="section section--bg" data-bg="img/bg/section__bg.jpg">
  <div class="container">
    ...
  </div>
</section>

section--first – section with margin-top

<section class="section section--first">
  <div class="container">
    ...
  </div>
</section>

section--dark – section with dark background color

<section class="section section--dark">
  <div class="container">
    ...
  </div>
</section>

section--bt – section with border-top

<section class="section section--bt">
  <div class="container">
    ...
  </div>
</section>
